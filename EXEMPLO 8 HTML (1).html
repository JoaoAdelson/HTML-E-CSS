Elemento de imagem img

O elemento Html que represenata imagens é a tag < img >. essa tag é usada para exibir imagens, Para utiliza-la podemos usar imagens dos mais variados formatos, para isso devemos usar os seguintes atributos: src, que representa o caminho da imagem e alt, que represeta um texto alternativo, que será carregada quando a imagem não estiver disponivelk
