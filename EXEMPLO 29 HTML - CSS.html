Exemplo 29 HTML CSS - Extilos aplicados ao Texto

Uma das principais aplicações do CSS é a estilização de textos, são muitas propriedades que podem ser aplicadas. Neste exemplo iremos aplicar algumas delas.

A tipografia é a arte de organizar o texto para torna-lo mais atraente e claro. Neste caso podemo usar várias propriedades CSS para alterar e deixa-la mais atraente, tais como: Font-family, text-aling, etc.
