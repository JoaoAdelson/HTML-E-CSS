Exemplo 31 HTML e CSS - BACKGROUND

Background refere-se em HTML ao fundo dos elementos. Ele pode ser aplicado de diferentes formas, por exemplo: mudando a cor de fundo, colocando imagens em diferentes posiçôes e tamanhos.

BACKGROUN IMAGE

Neste caso a imagem é usada como plano de fundo do elemento. Neste exemplo vamos usar uma imagem nesta div como fundo da div. O BACKGROUND IMAGE pode ser usado para personalizar toda a página ou apenas parte dela.

BACKGROUND REPEAT

A propriedade BACKGORUND REPEAT é utilizada de forma que se tenha o controle de repetição da imagemnos eixos x e y.
