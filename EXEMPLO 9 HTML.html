O elemento html main é uma tag semántica que define o conteúdo principal de uma página. Ela deve conter o conteúdo mais relevante para o usuário.

HTML Semântico

O html semântico é conceito do html que se refere ao uso de elementos que possuem significado claro e descritivo, tanto para desenvolvedores quanto para navegadores. Além disso, eles são essenciais para os mecanismos de busca e leitores de tela.

Neste exemplo estamos utilizando a tag semântica section, essa tag agrupa contéudo relacionados dentro da tag main. Dentro da tag section poderemos ter também a tag article, essa tag representa um conteúdo independente, como posts de blog e notícias.

Tag de valor semântico neutro - <div>

A tag <div>é aquela que possui op valor semântico neutro. Ela é muito utilizada para agrupar elemento sem um significado específico. Geralmente a utilizamos quando queremos criar um layout.
