Formulários

Os formulários em html são usados par coletar informações dos usuários, como login, cadastro, pesquisas eenvio de dados. O elemento principal é o <form>, que agrupa os campos de entrada

Método de Get em formulários HTML

O método GET é uma das formas de envio de ddos em formulários HTML, sendo utilizado para transmitir informações através da url. Ele é definido no atributo method do form é o método padrão caso nenhum seja específicado.
