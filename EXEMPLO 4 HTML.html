Exemplo 4 - HTML (Estrutura Basica de um documnento HTML)

Este exeplo mostra a Estrutura minima de um documento HTML. Aqui estao as principais tags utilizadas:

  • <!DOCTYPE html>:Define o tipo da versão do documento Html. No nosso caso a versõo atual é o HTML5
  • <Html html>: É o elemento central do documento, é o conteiner raiz de toda a página
  • <head html>: Contém metadados, como o titulo da página e links para arquivos externos.
  • <body html>: Parte visivel da pagina. Tudo oque estiver dentro dessa tag ficará visível.
