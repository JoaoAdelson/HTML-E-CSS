Exemplo 10 Html - Elemnto tabela

Para criarmos tabelas em documentos HTML devemos usar <TABLE>, essa tag permite que possamos organizar nossos dados em linhas e colunas. Cada tabela é composta por uma estrutura de linhas, cabeçalho e colunas(celulas)

Descrição das principais TAGS que são utilizadas opara criar tabelas

  • <table >: Define a tabela
  • <tr - table row >:Define a linha da tabela - tr é o acrónimo de table row.
  • <th - table header >:Define o cabeçalho de uma coluna. th(table header)
  • <tb- table dada>:Define uma celula de dados(table dada)
