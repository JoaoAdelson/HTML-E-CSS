Exemplo 26 CSS - Tipos de vinculação de arquivos CSS

Existem três tipos básicos de vinculção de arquivos CSS, são elas: vinculções externas e as internas que se dividem em duas incorporadas e inline.

Neste exemplo, nós estamos utlizando a vinculação externa, que é a mais indicada para projetos maiores

Pra vincular arquivos CSS de forma externa nós usamos a tag LINK

Exemplo de aplicação da vinculação externa.

Como dito acima, estamos vinculando um arquivo CSS externo. Esse arquivo foi/será criado na mesma pasta onde está o documento HTML, no caso exemplo 26.
