Exemplo 21 HTML - CSS

CSS é um acrônimo para folhas de estilo em cascata. Essa técnica visa personalizar, estilizar um odumento HTML. Para isso, usamos regras CSS que representam uma estrutura que define como os elementos selecionados devem ser estilizados. Essa regra é composta pelas seguintes partes

  • Seletor: Indica quais elementos serão afetados pelo estilo definido.
  • Bloco de declaração: Neste caso o bloco é composto por propriedades e valores, estes elementos são aplicados aos elementos HTML.

Seletores de tipo

Como vimos anteriormente, os seletores são padrões usdos para selecionar elementos HTML e consequentimente aplicar regras de estilo

O seletor de título é caracterizado por utilizar nomes de tags HTML. Sendo aplicado a um ou mais tipos de elementos de referência

Neste exemplo estamos utilizando seletores de tipo para estilizar o documento HTML. Para informações acesse a estrutura dessa página.
