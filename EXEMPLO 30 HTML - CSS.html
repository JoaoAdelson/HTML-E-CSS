Exemplo 30 HTML CSS - Cores

Quando falamos de cores em CSS, estamos falando de uma infinidade de aplicações, que podem ser definidas de diversas formas. A seguir vamos dscrever cada uma delas.

RGB/RGBA

Essas cores são baseadas na combinação de cores obtidas a partir do vermelho; Verde; Azul. Além disso, podemos ainda adicionar uma nova variante "a", o "a" ele se refere ao nível de transparência, e vai de 0 a 1. Quanto mais prórximo do 1 menos transparente ele será.

Cores Hex

No caso das cores baseadas no sistema hexadecimal as cores são formadas combinando o # com números que vão do 0 ao 9e da letra A até o F. O sistema hexadecimal é muito comum em documentos HTML/CSS.

HSL/HSLA

O sistema de cores HSL é aquele que se baseia na MATIZ - HUE e é fornedido em graus no circulo cromático. Temos ainda o S de saturaçãop e o L de luminozidade
