Exemplo 32 HTML e CSS - UNidades de medida

Unidades de medida em CSS determinam tamanhos, espaçamentos, larguras, alturas, tamanhos de fontes, etc. Elas dizem o quão grande ou quão longe um elemento pode ser. Neste caso temos duas unidades básicas de medidas, são elas: Absolutas e Relativas

UNIDADE DE MEDIDA - ABSOLUTA

Essas unidades de medida tem como caracteristica sempre apresentarem o mesmo tamanho, independete do contexto. Neste exemplo usando no texto desta DIV unidade de medida paica - pc.

UNIDADES DE MEDIDAS - RELATIVOS

As unidades de medidas relativas são aquelas que dependem de algum valor externo(como fonte, tamanho da página, ect.), ou seja do tamanho do elemento pai.

Podemos exemplificar as seguintes unidades de medida relativa: o em(Baseado no tamanho mèdio do m), rem(Baseado no root - r e no em), nestes dois exemplos que 1 em = 1 rem = 16px.
