Exemplo 6 HTML - Uso das tags para formatação de texto e organização visual.

Este exeomplo desmonstra o uso das tags para formatação de texto e consequentemente de sua oirganização visual.

Descrição

As tags abordadas:

  • <p >: Define um paragrafo.
  • <hr >: Cria um alinha horizontal, para separar conteudos.
  • <br >: Insere uma quebra de linha.
  • <Strong >: Define texto em negrito - com importância semantica.
  • <em >: Define o texto em itálico
